# Sales Analysis Dashboard (Excel Project)

## 📊 Project Overview

This project focuses on analyzing sales data using Microsoft Excel. An interactive dashboard is created to track key business metrics such as sales, profit, orders, and performance across cities, states, and categories.

The dashboard uses charts, KPIs, and slicers to provide clear and actionable insights.

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Graphs
* Data Cleaning
* Data Visualization

## 📈 Key Performance Indicators (KPIs)

* Total Sales: ₹65,300K
* Total Profit: ₹11,448K
* Profit Margin: 18%
* Total Orders: 25K

## 📊 Dashboard Insights

The dashboard provides the following insights:

* Top 5 Cities by Sales
* Sales by Category (Clothing, Electronics, Furniture, Grocery)
* Monthly Sales Trend Analysis
* Top 5 Products by Sales
* Payment Mode Distribution (UPI, Card, Cash, Net Banking)
* Filter by City, State, Category, and Month

## 🎯 Project Objective

The objective of this project is to perform data analysis on sales data and build an interactive dashboard to understand business performance, customer trends, and sales patterns.
